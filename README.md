# 📊 Streaming Service Data Analysis Dashboard

## 📌 Project Overview
This project analyzes streaming service user data using Microsoft Excel. The dashboard was created with Pivot Tables and Pivot Charts to visualize the distribution of users across different countries and generate meaningful insights.

---

## 🎯 Objective
- Analyze user distribution by country.
- Identify the country with the highest number of users.
- Create an interactive dashboard using Microsoft Excel.
- Present key business insights through charts and summary metrics.

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Analysis

---

## 📂 Dataset
The dataset contains user information, including:
- User ID
- User Name
- Country
- Join Date
- Last Login

Total Records: **1000**

---

## 📊 Dashboard Features
- Total Users KPI
- Top Country
- Highest Users
- Pivot Table
- Pivot Chart (Users by Country)
- Key Insights
- Summary Report

---

## 📈 Key Insights
- USA has the highest number of users (158).
- France and UK show strong user engagement.
- Germany and Australia have stable user distribution.
- India has the lowest number of users (116).
- A total of 1,000 users were analyzed across 7 countries.
